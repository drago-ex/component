
## Drago Components
Bootstrap components such as modal and offcanvas.

[![PHP version](https://badge.fury.io/ph/drago-ex%2Fcomponents.svg)](https://badge.fury.io/ph/drago-ex%2Fcomponents)


## Technology
- PHP 8.3 or higher
- composer
- node.js

## Installation
```
composer require drago-ex/components
```
